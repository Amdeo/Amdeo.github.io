redis 基础使用和常用命令

redis

Redis 是一个基于内存的 key-value 数据库，常用来做缓存、计数器、排行榜、消息队列等场景。

它的特点是：

  • 读写速度快
  • 支持多种数据结构
  • 支持持久化
  • 常用于高并发场景

常见数据类型

Redis 常见的数据类型有：

  • String
  • Hash
  • List
  • Set
  • Zset（有序集合）

其中最常用的是 String、Hash、List。
